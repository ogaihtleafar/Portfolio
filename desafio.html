HTML Documents

(voltar)

All HTML documents must start with a document type declaration: !DOCTYPE html. The HTML document itself begins with html and ends with /html. The visible part of the HTML document is between body and /body.

HTML Headings

(voltar)

HTML headings are defined with the h1 to h6 tags.h1 defines the most important heading. h6 defines the least important heading.

Example:

This is heading 1

This is heading 2

This is heading 3

This is heading 1

This is heading 2
This is heading 3

HTML Paragraphs

(voltar)

HTML paragraphs are defined with the p tag

HTML tags

(voltar)

On HTML we have a variety of tags and uses for them :

The font tag was used in HTML 4 to specify the font face, font size, and color of text

Text with a deleted part, and a new, inserted part:

My favorite color is blue red!

An abbreviation is marked up as follows:

The WHO was founded in 1948.

This text contains superscript text.

This text contains subscript text.

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

For this tag ol we have two different ordered lists (the first list starts at 1, and the second starts at 50):

  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk

Define important text in a document we use strong:

This text is important!

Highlight parts of a text we use mark :

Do not forget to buy milk today.

Mark up text that is set off from the normal prose in a document we use i tag:

Lorem ipsum is the most popular filler text in history.

The RMS Titanic, a luxury steamship, sank on April 15, 1912 after striking an iceberg.

Mark up a misspelled word with the u tag:

This is some mispeled text.
